# _printf Project

## Description
This project is a custom implementation of the printf function in C as part of the Holberton School curriculum. It handles formatted output and supports basic conversion specifiers.

## Requirements
- GCC compiler
- Ubuntu 20.04
- Allowed functions: write, malloc, free, etc.

## Installation
Clone the repository:

```bash
git clone https://github.com/nourafah1/holbertonschool-printf.git
cd holbertonschool-printf

```
##Example 
```c
_printf("Hello %s\n", "World");
_printf("Number: %d\n", 42);

```

## Testing
```bash
./printf

```
